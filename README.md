# AfroFuture Intelligence — Product Suite

React-based visual showcase of the AfroFuture Intelligence platform.

## Quick Start

```bash
npm install
npm run build
npm run deploy
```

## Deploy to GitHub Pages

1. Push code to GitHub
2. Run `npm run deploy`
3. Site live at `https://YOUR_USERNAME.github.io/afrofuture-intelligence/`

## Sections

- **Hero**: Stats, CTAs
- **Culture Report**: 4 markets, TOC, headline indicators
- **Intelligence API**: 5 endpoints, code blocks with brand sentiment data
- **Dashboard**: Brand bars, cultural rings, live event moat visualization
